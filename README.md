Analisador Léxico - Trabalho Acadêmico Sobre o Projeto Este repositório contém a implementação de um Analisador Léxico, desenvolvido como parte de um trabalho acadêmico. 

O objetivo do projeto é demonstrar a aplicação prática de Autômatos Finitos como mecanismos reconhecedores de Linguagens Regulares, utilizando uma matriz de transição de estados para analisar tokens fornecidos pelo usuário.

Funcionamento Durante a digitação de um token: O autômato reconhece cada símbolo inserido. A troca de estado ocorre conforme a matriz de transição de estados. Após a digitação de um espaço: O token é avaliado como reconhecido ou rejeitado, com base no estado final alcançado. 

Este projeto foi desenvolvido utilizando Delphi. A escolha dessa tecnologia permitiu a criação de uma interface simples, funcional e interativa, alinhada aos objetivos acadêmicos do trabalho. Por meio delas, foi possível implementar o Analisador Léxico e simular o funcionamento de um autômato finito.
